A generic utility class for interfacing with calibration databases.

This class provides a foundational layer for database operations, supporting both local SQLite and remote PostgreSQL databases. It manages SQLAlchemy engine creation, session management, and common CRUD (Create, Read, Update, Delete) operations for CalibrationORM objects.

It is designed to be extended by specific database implementations (e.g., LocalCalibrationDB, RemoteCalibrationDB).
